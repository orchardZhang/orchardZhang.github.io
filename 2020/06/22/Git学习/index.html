Git学习

2020-7-7

作用

管理版本迭代的工具，多人开发，协同开发。

版本控制分类

  • 本地版本控制，代码版本由开发者自己保存
  • 集中版本控制，代码版本保存在公司服务器上，需要定期备份服务器。代表工具SVN
  • 分布式版本控制，各个开发人员都拥有自己的版本控制信息，所有的版本信息都同步到每个用户，每个人都有全部代码。代表工具Git，Git可以直接看到更新了哪些代码和文件。

    下载

  • 官网:https://git-scm.com/
  • 镜像:https://npm.taobao.org/mirrors/git-for-windows/

卸载

反安装，删除环境变量
