JAVA基础-I/O流

2019-7-28

IO流

  • 把数据存储到文件，也可以从文件中读取数据，常见应用，文件复制等
  • 输入流，读数据
    FileReader 父类Reader
  • 输出流，写数据
    FileWriter 父类Writer

字符流

  • 字符输入流 Reader
  • 字符输出流 Writer

FileWriter

构造方法

  • FileWriter(String fileName)
    构造给定文件名的FileWriter对象.
  • **FileWriter(String fileName, boolean append) **
    构造FileWriter对象给出一个文件名与一个布尔值，指示是否附加（追加）写入的数据，默认是flase就是上一个构造方法
  • 如果没有绝对路径默认在workspace中该包中

    成员方法

  • void write(String str);写一个字符串数据
  • void write(String str,int index,int len);写一个字符串的一部分数据
  • void write(int ch);写一个字符数据，这里写int类型的数据好处就是可以写char类型的数据，也可以写char对应的int类型的数据’a’ 97
  • void write(char[] chs);写一个字符数组数据
  • void write(char[] chs,int index,int len);写一个字符数据的一部分数据
  • 实现数据的换行
    \n可以实现换行，但是window系统自带的笔记本打开并没有换行，
    Windows识别的换行是\r\n ，mac换行\r,Linux换行\n；
  • 实现数据的追加写入

    FileReader

  • 从文件中读取数据并显示到控制台

    构造方法

  • FileReader（String fileName);

    成员方法

  • int read();一次性读取一个字符
    返回的是该字符对应的int值，如果想得到字符，直接强制转换成char
    当读数据，数据的返回值是-1的时候就表示没有数据了，可用于循环判断条件
  • int read(char[] cbuf);一个读取一个字符数组的数据,返回的是实际读取到的实际长度（\r、\n 等分别算一个字符），实际读取长度是-1的时候就没有数据可以读取了
  • 文件的复制（读取加写入）

BufferedWriter类

  • 将文本写入到字符输出流中，缓冲字符，以便提供对单个字符、数组和字符串的有效写入。
  • 可以指定缓冲区大小，也可以接受默认大小。默认是足够大的用于大多数目的。

    构造方法

  • BufferedWriter(Writer out)
    创建一个使用默认大小输出缓冲区的缓冲字符输出流

    成员方法

  • void close()
    关闭流，冲洗它。
  • void flush()
    冲流。
  • void newLine() 缓冲流的特殊功能
    写行分隔符。这个换行符由系统决定 ，可以用于换行，不用wtrie（”\r\n”)的形式了
  • void write(char[] cbuf, int off, int len)
    写入一个字符数组的一部分。
  • void write(int c)
    写一个字符。
  • void write(String s, int off, int len)
    写入字符串的一部分。

BufferedReader

  • 从一个字符输入流中读取文本，缓冲字符，以便提供字符、数组和行的有效读取。

    构造方法

  • BufferedReader(Reader in)
    创建一个使用默认大小输入缓冲区的缓冲字符输入流。
  • BufferedReader(Reader in, int sz)
    创建一个使用指定大小的输入缓冲区的缓冲字符输入流。

    成员方法

  • void close()
    关闭流并释放与它相关联的任何系统资源。
  • Stream lines()
    返回一个 Stream，其中的元素是线从这 BufferedReader读。
  • void mark(int readAheadLimit)
    标记流中的当前位置。
  • boolean markSupported()
    告诉这是否流支持的mark()操作，它。
  • int read()
    读取单个字符。
  • int read(char[] cbuf, int off, int len)
    将字符读入一个数组的一部分。
  • String readLine() 缓冲流的特殊功能
    读一行文本。一次读取一行数据，但是不读取换行符所以输出要用println
    没有数据的话显示null，用于判断条件

缓冲流读写比基本流效率高，所以一般采用缓冲流的形式读写文件，缓冲流和基本流差别只是创建对象时的不同，其他方法一致

File类

  • 用来管理文件及目录
  • 文件目录路径名的抽象表示形式，file类的实例时不可变的，只是指向一个路径而已，即使没有存在这个路径也不会报错

构造方法

  • File(String pathname)
    将指定的路径名转成一个file对象

  • File(String parent, String child)
    根据是定的父路径和文件路径创建file对象

  • File(File parent, String child)
    根据是定的父路径对象和文件路径创建file对象，父路径的对象也可以使用匿名对象

常用方法

创建功能

  • boolean createNewFile() 当指定文件不存在时，创建文件并且返回true
  • boolean mkdir() 创建文件夹，指定文件夹不存在时，创建文件并返回true
  • boolean mkdirs()
    创建指定文件夹 ，当文件夹的目录不存在，则顺便一块创建了
  • 创建的时文件夹还是文件是看参数里的方法，而不是看参数

删除功能

  • boolean delete() 如果要删除的文件或者文件夹存在就删除并且返回true，否则返回flase，如果传入的参数有多层文件夹，则删除的最尾的文件夹；删除一个文件夹这个文件夹里面不能有子文件夹或者子文件

获取功能

  • String getAbsolutePath()
    返回此抽象路径名的绝对路径名的字符串 ，可以调用string的相关方法

  • File getAbsoluteFile()
    返回此抽象路径名的file对象形式，可以调用file相关方法

  • String getName()
    返回的名称的文件或目录的路径名表示的抽象。可以调用string的相关方法

  • String getParent()
    返回此抽象路径名的父路径名的字符串，如果没有指定父路径就显示null

  • File getParentFile()
    返回此抽象路径名的母抽象路径名，路径名或 null如果这不叫父目录。

  • String getPath()
    返回的是创建对象时的路径，路径是绝对路径还是相对路径取决于你创建对象时的创建的是什么

  • long lastModified()
    返回文件的抽象路径名表示上次修改时间。返回值是毫秒值可以用date类相关方法

import java.io.Serializable;

public class Student1 implements Serializable {

    /**
 * 不写这个ID就有可能出现错误；
 * 若你在写出文件后修改了这个类的成员变量；就会直接修改ID导致要写入的类和写出的类ID不匹配
 * 出现错误
 * 直接写出这个编号可以固定id；无论将来成员变量怎么变化都不会改变这个id
 */
private static final long serialVersionUID = 2011937288501817425L;
String name;
int age;


public Student1(String name, int age) {
    super();
    this.name = name;
    this.age = age;
}


@Override
public String toString() {
    return "Student1 [name=" + name + ", age=" + age + "]";
}

}

public class ObjectOutputStreamDemo1 {
public static void main(String[] args) throws IOException, ClassNotFoundException {
method1();
//创建输入流对象
ObjectInputStream ois = new ObjectInputStream(new FileInputStream(“a.txt”));
//读取数据
Object obj = ois.readObject();

        //向下转型，获取具体的子类对象
        ArrayList<Student1> list = (ArrayList<Student1>) obj;
        //获取每个对象并且遍历
        for (Student1 student1 : list) {
            System.out.println(student1);
        }
        //释放资源
        ois.close();
    }

    private static void method1() throws IOException, FileNotFoundException {
        //创建对象输出流对象

        list.add( new Student1("赵六", 35));

        oos.close();
    }
