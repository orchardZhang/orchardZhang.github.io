JAVA基础-Java程序流程控制

2019-7-15

选择结构

  • 有if、 if…else、if…else if…else 等
  • 条件运算符 : 变量=布尔表达式？ 表达式1：表达式2
  • 如果布尔表达式为真，把表达式1的值赋给变量，否则变量的值为表达式2的值。

循环结构

  • 首先对判断条件进行计算，为真这进入循环体，为假则退出循环
  • 先执行一次循环体再判断，至少执行一次循环体
  • 执行步骤，1->2->4->3->2->4->3……

跳转语句

  • break

1.在switch，碰到break 程序跳出switch结构
2.在嵌套循环中碰到break则跳出内循环
3.终端当前循环执行下一个语句
4.case穿透：如果没有遇到break那么不论下一个case是什么，都会执行代码直到遇见下一个break

  • continue
    程序运行到continue语句时会停止运行剩余的循环语句，然后回到循环开始处继续运行。

递归

  • 通过不断调用自己来实现一定的功能
  • 注意：1.递归一定要有出口否则会内存溢出；
  • 2.递归的次数不宜过多，否则也会内存溢出；
