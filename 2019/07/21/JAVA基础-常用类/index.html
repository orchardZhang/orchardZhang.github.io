JAVA基础-常用类

2019-7-21

System类

  • 被final修饰，System类包含一些有用的类的字段和方法。它不能被实例化。
  • system方法全被静态static所修饰，直接调用类名即可

arraycopy(Object src, int srcPos, Object dest, int destPos, int length)

  • static void arraycopy(Object src, int srcPos, Object dest, int destPos, int length)
    从指定的源数组中复制一个数组，开始在指定的位置，到目标数组的指定位置。
  • 参数1：源数组；参数2：源数组的起始索引位置；参数3：目标数组；参数4：目标数组的起始索引位置；参数5，指定接收的元素个数

currentTimeMillis()

  • 返回当前系统时间以毫秒为单位
  • 这里的时间是相对值，相对于1970年1月1日0时刻的毫秒值

可以用于查看一个运行的效率

exit(int status)

  • public static void exit(int status)
    终止当前正在运行的java虚拟机。该参数作为状态代码；按惯例，一个非零状态码表示异常终止

static void gc()

  • public static void gc()运行垃圾收集器。虚拟机会自动调用
  • java虚拟机消耗的努力回收未使用的对象为了让记忆他们目前占据可以快速重用。当控制从方法调用返回，java虚拟机已经尽了最大努力去回收废弃物品的空间。

Date类(util)

  • 表示特定的瞬间，精确到毫秒，可以同坐方法来设定自己所表示的时间，可以表示任意的时间

构造方法

  • Date()
    分配一个 Date对象并将它初始化，据1970-1-1据现在毫秒数，显示的系统当前时间
  • Date(long date)
    分配一个 Date对象并将它初始化为代表指定的毫秒数自基准时间被称为“时代”，即1970年1月1日，00:00:00 GMT。

成员方法

  • 毫秒值转Date

  • void setTime(long time)
    代表一个时间点是1970年1月1日00:00:00 GMT time毫秒后这 Date对象，给一个毫秒值转成时间格式

  • Date转成毫秒值

  • long getTime()
    返回毫秒数，自1970年1月1日以来，该 Date对象表示00:00:00 GMT。

DateFormat类

SimpleDateFormat

  • 格式化 Date—->String
  • String format(Date date)
  • 解析 String –>Date
  • Date parse(Sring source)
  • 构造方法

1.SimpleDateFormat();使用默认的模式进行对象的构建
2.SimpleDateFormat(String patttern);使用特定的模式进行对象的构建

  • 默认模式
  • 用户自定义模式

**注意：格式化和解析的格式必须相同

  • 静态方法构造对象

  • 由于基本类型只能做一些简单的运算，所以Java为我们封装了基本数据类型；为每种基本数据类型提供了包装类
  • 包装类就是封装了基本数据类型的类，为我们同提供了更多的方法和一些变量；
