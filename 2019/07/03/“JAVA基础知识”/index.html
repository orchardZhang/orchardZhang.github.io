JAVA基础-JAVA语言基础

2019-7-3

Dos命令

  • 跳转盘 盘符 ：
  • 显示当前文件夹目录名称 dir
  • 进入某一个目录 cd 目录名称
  • 退回上一个目录 cd..
  • 一次进入两个目录 cd 目录1\\目录2
  • 多级回退 cd\
  • 清屏 CLS
  • 退出 exit

JAVA命名规范

  • 类名 类名尽量用名词，首字母大写，若由多个单词组成，则每个单词的首字母都要大写，例如DataOutputStream。
  • 接口名 与类名类似，但是一般用形容词，例如Runnable
  • 方法名和变量名 第一个单词全部小写，后续单词首字母大写，例如getProperties
  • 包名 由小写字母组成，一般采用域名的反写。例如 com.baidu
  • 常量名 所有字母全部采用大写，单词与单词之间采用_隔开，例如MAX_VALUE

WIN下记事本写JAVA

  • 源文件(.java)–>编译器–>字节码文件（.class）–>解释器–>机器码
  • 程序的编译和执行需要使用到javac和java命令，所以编写源文件要放在JDK的bin目录下(配置环境变量后则不用）
  • JAVA源文件名必须和类名一致

注释

  • 单行注释 //注释文字
  • 多行注释 /* * 注释文字 * */
  • 文档注释 /**注释文字 */
  • 作用：解释说明程序，提高程序的可读性。

关键字

  • 组成关键字的字母全部小写

标识符

  • 由数字、字母、_、$、不能由数字开头
  • 区分大小写
  • 不能为关键字

基本数据类型

  • JAVA数值类型占用的空间和范围与具体的机器无关
  • char 16位
  • byte 8位
  • short 16位
  • long 64位 定义long类型最好都在数字后面加一个L
  • int 32位
  • float 32位
  • double 64位
  • boolean – 布尔型 值为ture或false ,整型值和布尔值之间不能互相转换

常量

  • 整数常量 十进制、0x十六进制、0八进制
  • 浮点型常量 13.14、1.55e-2 默认为double型，若要为float型，在数值后面加f或F、13.14f
  • 字符型 单字符：’L’，转义字符 ASCII前32个控制字符。
  • 字符串常量 JAVA要求一个字符串在一行内写完，若需要多于一行的字符串，则需要用“+”联系起来。”Do one thing at a time”+”and do well.”
  • final double CN_PER_INCH=2.54;final关键字表示这个变量只能被赋值一次，一旦被赋值了就不能够在更改了

    变量

  • 语法格式： 数据类型 变量名1，变量名2；或 数据类型 变量名=初始值；
  • 变量赋值格式： 变量名=值； 或 变量名 = 表达式 ；

数值类型之间的强制转换

  • （要转换的数据类型）（表达式） 例如 （int）（3.1415926）。

    算术表达式

  • 双目运算符 + - * / %
  • 单目运算符 ++ – -（取反）

    赋值表达式

  • 变量名= 表达式
  • 复合赋值运算符+= 、-=、*=、/=、%=、&=、^=、|=

    关系表达式

  • 结果都是布尔型
  • ==、！=、<、>、<=、>=

    逻辑表达式

  • 结果为布尔值
  • & && | || ！ ^(异或）
  • &、&&区别 &：不管怎么样都会运行&号两边，&&：仅当&&前为真是才会运行&&后一个（称为短路求值）
  • 911！=911&911/0==0 ：非短路与，左右两个都要执行，执行到第二个除数为0，程序报错；
  • 911！=911&&911/0==0 ：短路与，左边不成立，结果值直接为0，不执行右边判断，整个狮子的值为0

    运算符的优先级

  • 没有必要全部记下，你要是可以通过小括号直接改变优先级

    Unicode字符集

  • 一个字符占2B
  • Unicode查询链接
  • 字符串 子串：String类的substring方法可以从一个较大的字符串中提取一个子串，
    String greeting = “Hello!”;
    String s =greeting.substring(0,3);
    创建了一个由字符Hel组成的字符串
  • s.equals(t):检测字符串s与字符串t是否相等（区分大小写）
    也可以是字符串常量 “hello”.equal(greeting)
  • 不区分大小写 equalsIgnoreCase
