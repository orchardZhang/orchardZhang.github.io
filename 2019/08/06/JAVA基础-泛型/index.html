JAVA基础-泛型及常见简单数据结构

2019-8-7

泛型的概述和体现

  • 由于集合可以存储任意类型的对象，当我们储存了不同类型的对象，就有可能在转换的时候出现类型转换异常，为了避免这个问题，Java给我们提供了一种机制，叫做泛型
  • 泛型：是一种广泛的类型，把明确数据类型的工作提前到了编译时期，借鉴了数组的特点
  • 泛型的好处：避免了类型转化的问题、可以减少黄色警告线、可以简化我们代码的书写
  • 使用时机：问API.当我们看到的时候就可以使用泛型了

foreach循环

  • 增强for循环，一般用于遍历集合或数组。
  • 注意，在增强for循环中不能修改集合，否则会出现并发修改异常，因为增强for循环的底层就是迭代器

常见数据结构

数组

  • 查找快，增删慢，因为数组的长度是确定的，一般情况下增删数组必须新建一个数组，然后把新建的数据插入到原先的数组当中去，效率较低
  • 常见的有ArrayList

链表

  • 有链子连接起来的一堆结点
  • 结点： 地址值，值（内容），下一个节点的地址值
  • 链表查找没有什么好的方法，只能遍历后一个一个地查看
  • 链表增删很简单，比如说要把结点2后面增加一个新的节点8，只需要把结点2的下一个结点的地址值修改成结点8的地址值，然后把结点8的下一个地址值修改成结点3的地址值即可，删一个直接该地址就行
  • 查询慢增删快
  • 常见的有LinkedList

栈

  • 先进后出，后进先出

队列

  • 先进先出，可以看成有两个出口的栈，类似于现实生活中的队列

遍历方式

  • 有索引的有四种1.for 2.迭代器 3.toArray() 4.foreach
