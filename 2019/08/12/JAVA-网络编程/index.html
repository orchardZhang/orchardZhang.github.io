JAVA-网络编程

2019-8-12

  • Socket：用于表诉IP地址和端口，再Internet的主机上一般运行了多个服务软件，同时提供几种服务，每一个服务都打开一个socket并且绑定到一个端口上，不同的端口对应不同的服务

  • soket就是为网络编程提供的一种机制；通信的两端都有socket；网络通信其实就是socket间的通信：数据在两个socket间通过IO传输

  • 网络通信三要素

  • IP地址：网络中设备的信号，点分十进制表示法

  • 端口号：物理端口 网卡口
    逻辑端口 表示进程的逻辑地址不同进程的表示

  • 传输协议：UDP协议，TCP协议

InetAddress

  • 这个类代表一个互联网协议（IP）地址，IP地址是一个32位(IPV4)或128位(IPV6)无符号数使用的IP

UDP

  • 使用udp收发数据的注意事项
  • 接收端和发送端的端口号要一致，端口号错误数据会发出但是不会出现异常，
  • 端口号已经绑定了就被占用了，所以端口号不能重复

DatagramSocket

  • 此类表示用来发送和接收数据，基于udp协议
  • 构造方法
  • DatagramSocket() 创建socket对象并随机分配端口号
  • DatagramSocket(int port) 创建socket对象并指定端口号
  • 成员方法

DatagramPacket

  • 这类表示一个数据报包。用于数据的打包
